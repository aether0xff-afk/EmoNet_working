좋아. 지금까지 나온 내용을 바탕으로 **네 감정 전달 뉴런 모델 전체를 일관된 형태로 쭉 정리**해볼게.
설명만 흩어져 있으면 나중에 구현할 때 꼬이니까, **개념 → 동작 흐름 → 수식 추천 → 타입별 기능 → 구현 시 주의점 → 미정/추가 논의 필요** 순서로 정리하겠어.

# 감정 전달 뉴런 모델 정리

## 1. 모델 개요

이 모델은 일반적인 인공신경망처럼 단순히 입력을 받아 출력만 만드는 것이 아니라,
**감정 벡터(`emo_vec`)를 전달하고**,
그 과정에서

* 발화 여부를 판단하고
* 감정을 기억으로 저장하고
* 과거 감정을 현재 감정에 반영하며
* 뉴런 종류에 따라 시냅스를 제거하거나 생성하고
* 심지어 네트워크 전체의 민감도까지 조절하는

**감정-기억-가소성 결합 뉴런 모델**이다.

---

## 2. 감정 벡터 정의

감정은 `emo_vec` 이라는 4차원 벡터로 표현한다.

```python
emo_vec = [dopamine, serotonin, norepinephrine, melatonin]
```

각 성분의 의미는 다음과 같다.

* **도파민**: 흥분성 강화, 새 연결 생성, 감정 편차 증폭
* **세로토닌**: 억제성 강화, 연결 제거, 감정 평준화
* **노르에피네프린**: 전체 네트워크 각성 증가, 발화 임계막전위 감소
* **멜라토닌**: 전체 네트워크 둔화, 일부 뉴런 일시적 드롭아웃

---

## 3. 뉴런의 기본 발화 규칙

각 뉴런은 입력 시냅스들로부터 신호를 받아 **입력 막전위 총합** `V_in` 을 계산한다.

```python
V_in = sum(weight_i * input_i)
```

그 후 `V_in` 이 해당 뉴런의 **발화 임계막전위** `V_th` 보다 크면 뉴런은 발화한다.

```python
if V_in > V_th:
    fire = True
else:
    fire = False
```

발화한 경우, 아래 순서대로 기능을 수행한다.

1. 공통 기능 수행
2. 뉴런 종류별 기능 수행
3. 최종 `emo_vec` 또는 조절 결과 반환

---

## 4. 공통 기능

공통 기능은 뉴런 종류와 상관없이, **발화한 모든 뉴런이 공통적으로 수행하는 기능**이다.

## 4-1. 기억 저장

입력 막전위 총합 `V_in` 이 **기억 임계막전위** `V_mem` 보다 크면, 현재 감정 상태를 기억으로 저장한다.

저장 형태는 다음과 같다.

```python
(memory_emo_vec, saved_timestep, saved_membrane_potential)
```

즉 저장되는 것은 다음 3가지이다.

* 당시의 `emo_vec`
* 저장 시점의 타임스텝
* 저장 당시의 막전위

이 규칙의 의미는 다음과 같다.

* 모든 감정을 다 기억하는 것이 아니라
* **충분히 강한 입력**만 기억으로 남긴다
* 그래서 감정적으로 강렬한 사건만 나중에 다시 영향을 준다

---

## 4-2. 과거 감정과 현재 감정의 합성

현재 시점에서는 저장된 과거 기억들을 다시 불러와서, 현재 감정과 합성한다.

단, 과거 감정은 그대로 더하는 것이 아니라 다음 요소를 반영한다.

* 저장 당시의 막전위
* 현재 시점과 저장 시점의 차이에 따른 시간 감쇠
* 마지막 정규화

### (1) 시간 차이 계산

현재 시점이 `t_now`, 기억 저장 시점이 `t_k` 이면

```python
delta_t = t_now - t_k
```

### (2) 감쇠 계수

기억은 시간이 지날수록 약해지므로 지수 감쇠를 사용한다.

```python
decay_k = exp(-lambda_ * delta_t)
```

여기서 `lambda_` 는 감쇠 강도이다.

* 작을수록 오래 기억됨
* 클수록 빨리 잊힘

### (3) 각 기억의 기여도

저장된 감정 벡터 `E_k` 와 저장 당시 막전위 `V_k` 를 사용하여:

```python
contrib_k = E_k * V_k * decay_k
```

즉,

* 강한 막전위로 저장된 기억일수록 영향이 크고
* 오래된 기억일수록 영향이 줄어든다

### (4) 전체 기억 합산

```python
memory_sum = Σ(E_k * V_k * decay_k)
```

### (5) 현재 감정과 합성

현재 감정 `E_now` 와 과거 기억의 합 `memory_sum` 을 가중합한다.

```python
E_mix = w_now * E_now + w_mem * memory_sum
```

여기서:

* `w_now`: 현재 감정 비중
* `w_mem`: 과거 기억 비중

### (6) 정규화

합성 후 벡터가 너무 커지거나 해석이 어려워지는 걸 막기 위해 정규화한다.

추천 방식은 **L1 정규화**이다.

```python
def normalize(vec, eps=1e-8):
    vec = [max(0, x) for x in vec]
    s = sum(vec)
    if s < eps:
        return [0.25, 0.25, 0.25, 0.25]
    return [x / s for x in vec]
```

최종적으로:

```python
E_final = normalize(E_mix)
```

공통 기능 수행 후, 뉴런은 자신의 종류에 따라 별도 동작을 수행한다.

# 5-1. 억제성 뉴런

억제성 뉴런은 현재 감정 벡터의 **세로토닌 값**에 반응한다.

## 기능 1. 연결된 시냅스 제거

현재 뉴런과 연결된 시냅스들 중 일부를 **세로토닌 양에 비례해서 실제로 삭제**한다.

즉, 단순히 가중치를 낮추는 것이 아니라 **시냅스 자체를 없애는 방식**이다.

예를 들어 세로토닌 값 `s` 가 0~1 범위라고 하면:

```python
remove_ratio = k_remove * s
```

그리고 현재 연결된 시냅스 개수가 `N` 개라면:

```python
num_remove = int(N * remove_ratio)
```

그 후 무작위 또는 특정 기준으로 `num_remove` 개의 시냅스를 제거한다.

이 기능의 의미는:

* 연결 수 축소
* 정보 흐름 감소
* 시스템 안정화

---

## 기능 2. 감정 평준화

억제성 뉴런은 현재 `emo_vec` 의 각 성분을 평균 쪽으로 끌어당겨 감정 편차를 줄인다.

현재 벡터가 `E = [e1, e2, e3, e4]` 이고 평균이 `m` 이면:

```python
m = sum(E) / 4
```

평준화된 각 성분은 다음처럼 계산한다.

```python
e_i' = e_i + alpha * (m - e_i)
```

또는 같은 식으로:

```python
e_i' = (1 - alpha) * e_i + alpha * m
```

여기서 `alpha` 는 평준화 강도이고, 세로토닌에 비례하게 둔다.

추천:

```python
alpha = min(0.8, 0.6 * serotonin)
```

의미:

* `alpha = 0` 이면 변화 없음
* `alpha = 1` 이면 전부 평균이 됨
* 세로토닌이 높을수록 감정 편차가 줄어듦

마지막에 다시 정규화해서 반환한다.

```python
E_out = normalize(E_flattened)
```

# 5-2. 흥분성 뉴런

흥분성 뉴런은 현재 감정 벡터의 **도파민 값**에 반응한다.

## 기능 1. 새로운 시냅스 생성

현재 뉴런에 새로운 시냅스를 **도파민 양에 비례해서 랜덤하게 생성**한다.

도파민 값이 `d` 이면:

```python
new_ratio = k_new * d
num_new = int(max_new_synapses * new_ratio)
```

그리고 네트워크 내 연결 가능한 다른 뉴런들 중 일부를 랜덤하게 골라 새 연결을 만든다.

이 기능의 의미는:

* 연결 확장
* 정보 전달 경로 증가
* 학습/적응 경로 다양화

---

## 기능 2. 감정 반평준화

흥분성 뉴런은 `emo_vec` 의 각 값을 평균에서 더 멀어지게 만들어 감정 대비를 증폭시킨다.

평균 `m` 에 대해:

```python
e_i' = e_i + beta * (e_i - m)
```

여기서 `beta` 는 반평준화 강도이다.

추천:

```python
beta = min(1.0, 0.8 * dopamine)
```

이 식의 효과는:

* 평균보다 큰 값은 더 커짐
* 평균보다 작은 값은 더 작아짐
* 감정 대비가 강화됨

이후 값이 폭주하지 않도록 다음 안전장치를 둔다.

### 안전장치 1: 음수 방지

```python
e_i' = max(0, e_i')
```

### 안전장치 2: 정규화

```python
E_out = normalize(E_sharpened)
```

# 5-3. 조절성 뉴런

조절성 뉴런은 특정 뉴런 하나의 출력만 바꾸는 것이 아니라,
**신경망 전체 상태를 전역적으로 조절**하는 역할을 한다.

## 기능 1. 멜라토닌 기반 드롭아웃

현재 `emo_vec` 의 멜라토닌 값에 비례해 전체 뉴런 중 일부를 **이번 사이클에만 비활성화**한다.

멜라토닌 값 `mela` 에 대해:

```python
drop_ratio = k_drop * mela
num_drop = int(total_neurons * drop_ratio)
```

그 후 무작위로 `num_drop` 개 뉴런을 골라 이번 턴 동안만 dropout 상태로 둔다.

중요:

* 이 dropout 은 **영구적이 아님**
* **다음 사이클이 시작되면 초기화됨**

의미:

* 고멜라토닌 상태에서 활동 억제
* 휴식/둔화 모드 구현

---

## 기능 2. 노르에피네프린 기반 임계막전위 감소

현재 `emo_vec` 의 노르에피네프린 값에 비례해 **모든 뉴런의 발화 임계막전위**를 낮춘다.

노르에피네프린 값 `n` 에 대해:

```python
delta_th = k_ne * n
V_th_all = V_th_all - delta_th
```

단, 너무 낮아져서 모든 뉴런이 무조건 발화하면 안 되므로 하한값을 둔다.

```python
V_th_all = max(V_th_min, V_th_all - delta_th)
```

의미:

* 노르에피네프린이 높을수록
* 뉴런이 더 쉽게 발화하고
* 전체 네트워크가 더 민감해짐

즉 각성/경계 상태를 나타냄.

---

## 기능 3. 특정 감정이 압도적일 때 기억 임계막전위 감소

현재 `emo_vec` 에서 특정 성분 하나가 유난히 크다면, 그 순간은 감정적으로 강렬하다고 보고 **기억 임계막전위**를 낮춘다.

그 결과 그 순간의 입력은 더 쉽게 기억으로 저장된다.

### 압도적 감정 판정 기준 추천

현재 벡터에서 최대값을 `e_max`, 전체 합을 `total`, 평균을 `m` 이라 하자.

```python
total = sum(E)
m = total / 4
e_max = max(E)
```

다음 두 조건을 동시에 만족하면 “압도적”이라고 판정한다.

```python
(e_max / total >= 0.5) and (e_max >= 1.8 * m)
```

즉:

* 한 성분이 전체 감정의 50% 이상을 차지하고
* 평균보다 1.8배 이상 크면
* 특정 감정이 지배적이라고 본다

이 경우:

```python
V_mem = max(V_mem_min, V_mem - delta_mem)
```

이렇게 기억 임계막전위를 낮춘다.

의미:

* 매우 강한 감정은 더 쉽게 각인됨
* 감정적 사건이 기억에 오래 남는 효과 구현

---

## 6. 전체 동작 흐름

이 모델의 한 뉴런이 한 타임스텝에서 동작하는 전체 순서는 다음과 같다.

### 단계 1. 입력 막전위 계산

```python
V_in = sum(weight_i * input_i)
```

### 단계 2. 발화 여부 판단

```python
if V_in <= V_th:
    발화하지 않음
    종료
```

### 단계 3. 공통 기능 수행

* `V_in > V_mem` 이면 현재 `(emo_vec, timestep, V_in)` 저장
* 저장된 과거 감정들을 시간 감쇠와 함께 불러옴
* 과거 감정과 현재 감정을 정규화 기반으로 합성

### 단계 4. 뉴런 타입별 기능 수행

* 억제성:

  * 세로토닌 비례 시냅스 삭제
  * `emo_vec` 평준화
* 흥분성:

  * 도파민 비례 새 시냅스 생성
  * `emo_vec` 반평준화
* 조절성:

  * 멜라토닌 비례 일시적 dropout
  * 노르에피네프린 비례 전체 임계막전위 감소
  * 특정 감정 지배 시 기억 임계막전위 감소

### 단계 5. 최종 출력 반환

* 수정된 `emo_vec`
* 또는 네트워크 상태 변경 결과 반영

* **기억 시스템**:

## 9. 간단한 의사코드

아주 거칠게 쓰면 이런 구조야.

```python
def neuron_step(neuron, inputs, emo_vec, t_now, network):
    V_in = compute_membrane_potential(inputs)

    if V_in <= neuron.fire_threshold:
        return None

    # 1. 기억 저장
    if V_in > neuron.memory_threshold:
        neuron.memory.append((emo_vec, t_now, V_in))

    # 2. 과거 기억과 현재 감정 합성
    memory_sum = [0, 0, 0, 0]
    for E_k, t_k, V_k in neuron.memory:
        delta_t = t_now - t_k
        decay = exp(-lambda_ * delta_t)
        memory_sum += E_k * V_k * decay

    E_mix = w_now * emo_vec + w_mem * memory_sum
    E_mix = normalize(E_mix)

    # 3. 타입별 기능
    if neuron.type == "inhibitory":
        remove_synapses(neuron, ratio=k_remove * E_mix[1])  # serotonin
        E_out = flatten_toward_mean(E_mix, alpha=min(0.8, 0.6 * E_mix[1]))

    elif neuron.type == "excitatory":
        add_random_synapses(neuron, network, ratio=k_new * E_mix[0])  # dopamine
        E_out = sharpen_from_mean(E_mix, beta=min(1.0, 0.8 * E_mix[0]))

    elif neuron.type == "modulatory":
        apply_dropout(network, ratio=k_drop * E_mix[3])  # melatonin
        decrease_global_thresholds(network, amount=k_ne * E_mix[2])  # norepinephrine

        if is_dominant(E_mix):
            neuron.memory_threshold = max(mem_min, neuron.memory_threshold - delta_mem)

        E_out = E_mix

    return normalize(E_out)
```

## 10. 이 모델의 핵심 특징

이 모델의 핵심은 단순한 감정 전달이 아니라 다음 4가지를 동시에 한다는 점이야.

### 1. 감정 전달

현재 `emo_vec` 을 다음 뉴런/시스템에 전달한다.

### 2. 감정 기억

강한 감정을 저장해 이후에도 영향을 주게 한다.

### 3. 구조 가소성

감정 상태에 따라 시냅스를 제거하거나 새로 만든다.

### 4. 전역 조절

특정 감정 성분은 네트워크 전체의 민감도와 활동량을 조절한다.

그래서 이건 그냥 “감정값 붙은 뉴런”이 아니라
**감정이 신경망 구조와 상태를 바꾸는 동적 시스템**이라고 볼 수 있어.

# 미정 / 추가 논의 필요

아래는 지금 구조는 잡혔지만, 나중에 구현할 때 꼭 더 정해야 하는 부분들이야.

## 1. 시냅스 삭제/생성 대상 선정 방식

* 완전 랜덤으로 할지
* 최근 잘 안 쓰인 연결부터 지울지
* 비슷한 감정을 주는 뉴런끼리 우선 연결할지

이건 성능과 해석에 꽤 큰 차이를 만든다.

## 2. 기억 저장 개수 제한

기억을 무한히 저장하면 계산량이 커진다.
그래서

* 최근 N개만 저장할지
* 너무 약해진 기억은 자동 삭제할지
  정해야 한다.

## 3. 감정 벡터 범위

각 값이

* 0~1 인지
* 0~100 인지
* 합이 1이 되도록 유지할지
  정의가 필요하다.

개인적으로는 **각 성분 0 이상, 전체 합 1** 구조가 제일 깔끔하다.

## 4. 조절성 뉴런의 전역 효과 지속 시간

* 한 턴만 적용할지
* 몇 턴에 걸쳐 서서히 복구할지
  정하면 더 자연스러워진다.

## 5. 기억 임계막전위 감소 폭

압도적 감정이 나왔을 때 얼마나 크게 낮출지에 따라
기억 저장 빈도가 확 달라진다.

## 6. 입력 막전위와 emo_vec의 관계

지금은 둘이 별도로 들어오지만,
나중엔 “특정 감정이 막전위 계산에도 직접 영향을 주게 할지”도 생각해볼 수 있다.
